# 겪은 문제점들
- router의 순환 참조 오류
    - 기존에 router에 모두 포함시켰던 기능들로 인해 코드의 가독성이 심각하게 저하되었다. 이를 해결하기 위해 기능 별 파일 분리가 필요하다고 느꼈다. 파일들을 분리하고 모듈 불러오기를 최상단에서 한번에 시행한 결과 router 순환 참조 오류가 발생하였다. 이를 해결하기 위해 router 변수 정의 후, 모듈 불러오기를 시행하였다. (ex : app\router\v1\main\router.py)
